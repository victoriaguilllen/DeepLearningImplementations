# DeepLearningImplementations

Collection of from-scratch implementations of core deep learning models and techniques.

## Included
- CNN
- CNN with Dropout
- RNN
- Batch Normalization
- MaxOut

## Goal
Understand how neural networks work internally by implementing them without high-level frameworks.

## Context
These implementations were developed as part of academic coursework and further extended for deeper understanding.

## Notes
Each module is self-contained and focuses on a specific architecture or technique.
